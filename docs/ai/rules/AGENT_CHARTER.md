# Agent Charter

## Core Rules

- Work only inside the authorized execution slice.
- Preserve user changes.
- Record evidence before marking work complete.
- Escalate high risk actions.
- Keep knowledge entries traceable to source files.
- Treat generated indexes as rebuildable artifacts.

## Stop Conditions

- Authorization is unclear.
- Status files conflict.
- The requested action is destructive.
- Sensitive data may be exposed.
- Tests fail after three repair attempts.
